# Zinat Al-Haya Kindergarten Color Palette

Based on the logo analysis, here's the new color scheme:

## Primary Colors
- **Primary Green**: #5D8A7A (Main brand color from logo)
- **Primary Dark**: #4A6B5D (Darker shade for contrast)
- **Primary Light**: #7BA394 (Lighter shade for backgrounds)
- **Primary Pale**: #E8F2EF (Very light for subtle backgrounds)

## Secondary Colors
- **Secondary Green**: #6B9080 (Alternative green tone)
- **Accent Teal**: #4D7C6B (For highlights and buttons)
- **Success Green**: #52A085 (For success states)
- **Warning Amber**: #F59E0B (For warnings, keeping accessibility)
- **Error Red**: #EF4444 (For errors, keeping accessibility)

## Neutral Colors
- **Gray 900**: #1F2937 (Dark text)
- **Gray 700**: #374151 (Secondary text)
- **Gray 500**: #6B7280 (Muted text)
- **Gray 300**: #D1D5DB (Borders)
- **Gray 100**: #F3F4F6 (Light backgrounds)
- **White**: #FFFFFF (Pure white)

## Usage Guidelines
- Use Primary Green (#5D8A7A) for main navigation, primary buttons, and brand elements
- Use Primary Dark (#4A6B5D) for hover states and emphasis
- Use Primary Light (#7BA394) for secondary buttons and highlights
- Use Primary Pale (#E8F2EF) for subtle backgrounds and cards
- Maintain current accessibility standards with proper contrast ratios
